# Results Directory

This directory contains outputs generated by the analysis pipeline.

## Expected Outputs

### `/figures/`
Visualization outputs from EDA and modeling:
- `target_distribution.png` - Target variable (y) distribution plot
- `equipment_boxplot.png` - Quality index by equipment category
- `duration_scatter.png` - Process duration vs quality scatter plot
- `sensor_correlations.png` - Top sensor correlation heatmap
- `feature_importance.png` - Model feature importance plot
- `model_comparison.png` - AutoML model comparison results
- `residual_plot.png` - Prediction residual analysis

### `/tables/`
Tabular results and statistics:
- `descriptive_stats.csv` - Feature descriptive statistics
- `vif_analysis.csv` - Variance Inflation Factor results
- `feature_selection.csv` - Selected features with MI scores
- `model_metrics.csv` - Model performance comparison (RMSE, MAE, R2)
- `hyperparameter_tuning.csv` - Bayesian optimization results

### Root Files
- `preprocessed_data.pkl` - Preprocessed dataset (X_train, y_train, X_predict)
- `best_model.pkl` - Trained best model for inference

## How to Generate

Run the notebooks in order:
```bash
# Step 1: Generate preprocessed_data.pkl
jupyter notebook 01_Code/01_eda_and_preprocessing.ipynb

# Step 2: Generate model outputs
jupyter notebook 01_Code/02_modeling.ipynb
```

## Note
Generated files (*.pkl) are excluded from git via `.gitignore`.
