===============================================================================
                        DATA INFORMATION

Data Source: SK HYNIX Semiconductor Manufacturing
Period: October 2021
Total Samples: 611 (train), 256 (test)

===============================================================================
                        FILE DESCRIPTIONS
===============================================================================

1. train_sensor.csv (~24.5MB)
   - Training sensor data in long format
   - Columns:
     * module_name: Equipment module ID (e.g., EQ4_PM6)
     * key_val: LOT identifier (e.g., LOT64_18)
     * param_alias: Sensor parameter name (e.g., EPD_para4)
     * mean_val: Average sensor reading
     * end_time: Process completion timestamp
     * step_id: Process step ID (4, 6, 12, 13, 17, 18, 20)

2. train_quality.csv (~25KB)
   - Training quality labels
   - Columns:
     * key_val: LOT identifier
     * msure_val: Quality measurement value (Target: y)
     * end_dt_tm: Measurement timestamp

3. predict_sensor.csv (~10.5MB)
   - Test sensor data (same format as train_sensor.csv)
   - Used for final predictions
